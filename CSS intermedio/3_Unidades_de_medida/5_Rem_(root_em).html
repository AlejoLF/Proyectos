Unidad de medida ROOT EM

ROOT EM actua exactamente igual que el EM nada mas que siempre hace referencia al font-size de la etiqueta RAIZ de la pagina, es decir a la etiqueta HTML.

Ej:

html {

font-size: 20px;

}

p {

font-size: 1.5rem;

}

En este caso el tama√±o tipografico de todas las etiquetas P estan haciendo referencia al font-size de la etiqueta raiz osea a la etiqueta HTML.

Dando por finalizado el capitulo de unidades de medida aconsejamos utilizar siempre unidades de medida relativas haciendo referencia siempre a una sola medida. Asi luego poder editar o cambiar las medidas mas facilmente en el futuro.
