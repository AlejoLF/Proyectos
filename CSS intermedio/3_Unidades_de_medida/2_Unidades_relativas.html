Unidades Relativas

En principio lo que hay que entender es que hoy en dia es cada vez mas importante tener en cuenta a muchos dispositivos y que un sitio web o una app, se puede ver en pantallas pequeñas como tablets o celulares, y tambien pantallas grandes como televisores o monitores.

Entonces si uno utiliza medidas absolutas, como puede ser ( 500px ) o ( 1024px), puede resultar ser una medida MUY grande para una pantalla de un celular, o una medida MUY pequeña para un monitor de 25 pulgadas.

Entonces vamos a ir adentrandonos en las medidas relativas para que nuestros sitios web se vean bien en cualquier dispositivo. Las medidas relativas que vamos a usar son el PORCENTAJE, EM y REM.
