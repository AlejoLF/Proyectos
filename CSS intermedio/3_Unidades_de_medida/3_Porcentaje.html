Porcentaje

La forma de manejar esta medida tiene que ver con el espacio que este elemento tiene disponible.

Ej:

ul {

width: 1000px;

}

li {

width: 50%;

}

Como al UL le dijimos de forma ABSOLUTA que tiene que medir 1000px de ancho, y a la etiqueta LI que se encuentra dentro le dijimos que su ancho tiene que ser del 50%. El ancho de la etiqueta LI va a ser de 500px. Ya que el espacio disponible que tiene es de 1000px ( el ancho de su etiqueta "padre" ),

Esta misma idea puede usarse para los MARGENES, al indicar que el margin-right: 25%;, vamos a estar modificando de manera indirecta el tamaño del contenido ya que el espacio que resta va a ser para el contenido.

Ej:

ul {

width: 1000px;

}

li {

width: 50%;

margin-right: 25%;

}

Ahora si el UL pasa a medir 100% del ancho vamos a empezar a usar medidas variables y entonces cuando el usuario agrande o achique el navegador esas cajas van a empezar a achicarse o a agrandarse segun el tamaño de la pantalla.

Ej:

ul {

width: 100%;

}

li {

width: 50%;

margin-right: 25%;

}

Cuerpo Tipografico

Algo muy interesante es que el porcentaje puede usarse para el tamaño del cuerpo tipografico. Si nosotros le indicamos a un H1 que el tamaño de su cuerpo tipografico debe ser del 200% lo que va a pasar es que su cuerpo tipografico va a aumentar al doble del tamaño que tiene heredado.

Ej:

Si indicamos que el BODY va a tener su texto en tamaño de 16 px, y luego indicamos que los H1 van a tener un tamaño del 200%, siendo 16px su valor "heredado" el tamaño va a ser de 32px. Luego podriamos indicar que los H2 tengan un tamaño del 50%, etc:

body {

font-size: 16px;

}

h1 {

font-size: 200%;

}

h2 {

font-size: 50%;

}

A partir de este momento todos los tamaños del font-size dependen del font-size del body. Entonces si el dia de mañana queremos achicar o agrandar el font-size de todo el sitio, solamente modificamos el font-size del BODY.
