FIRST-CHILD

Ya veniamos hablando de Pseudo clases anteriormente. Hemos visto ya link/visited/hover/active como pseudo clases, y ahora vamos a amplicar un poco mas nuestra gama.

La pseudo class first-child nos permite diseñar distinto a cierto elemento que sea el "primer-hijo" de su "padre" es decir de la caja que los contiene. Ej:

p:first-child {

color: black;

}

De esta manera al tener varios parrafos pero queremos diseñar distinto solo al primer parrafo, lo que podemos hacer es añadir esta pseudo class y estamos seleccionando solo al primer parrafo que se encuentre dentro de su "padre", es decir su caja contenedora.
