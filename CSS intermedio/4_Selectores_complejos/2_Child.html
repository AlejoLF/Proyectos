Selector child

Este selector permite seleccionar, valga la redundancia, a todos los elementos " hijos" de una determinada etiqueta. A diferencia de la descendencia como puede ser seleccionar a todos los LI que se encuentren dentro de una lista DESORDENADA:

ul li {

padding: 30px;

}

A diferencia de esto.. el selector ">" (CHILD) selecionna a todas las etiquetas que sean directamente descendientes, es decir sus hijos, no sus nietos, ni sus bisnietos. Ej:

ul>li {

padding: 25%;

}

De esta manera estamos diseñando a los LI que desciendente directamente de una lista desordenada, si dentro de esa lista desordenada hay más listas con más LI, no los va a tener en cuenta.

div#contenido>p {

font-size: 25px;

}

Asi estamos seleccionando solo los P que sean hijos directos de un DIV con ID "contenido". Los P que se encuentren dentro de este mismo DIV con una profundidad de mas de 1 no se van a tener en cuenta.
