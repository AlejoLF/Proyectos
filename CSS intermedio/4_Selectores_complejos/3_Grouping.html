Grouping

Imaginemos que queremos darle el mismo diseño a varias etiquetas distintas. Para esto se usa el GROUPING, se agrupan distintas etiquetas con un mismo diseño. Esto se hace poniendo una "," (coma) luego de cada etiqueta, Ej:

h4, p.destacado, #principal, div>p, article p ul li {

background-color: #FF0000;

}

Esto nos ayuda a simplificar mucho mas las cosas y a no repetir tanto codigo.
