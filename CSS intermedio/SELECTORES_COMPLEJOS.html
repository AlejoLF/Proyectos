Repaso de selectores simples: descendientes, id, class.

El caso del selector mas simple es el que lleva solamente el nombre de la etiqueta que queremos diseñar:

p {

font-size: 2rem;

}

Otro caso es el del descendiente. Cuando colocabamos el nombre de una etiqueta seguido de otra con un espacio, indicabamos que esa etiqueta se encontraba dentro de la otra. Es decir, si quiero diseñar todos los LI que se encuentran dentro de una lista DESORDENADA:

ul li {

font-weight: bold;

}

Otra posibilidad es diseñar a un elemento particular. Ya sabemos que a los elementos en HTML se les puede poner un ID único y asi diferenciarlos del resto. En CSS para seleccionar ese elemento con determinado ID se coloca el nombre de la etiqueta a la cual queremos diseñar seguido del caracter # (numeral) y el nombre del ID:

ul#botonera {

min-width: 250px;

}

En este caso estamos seleccionando el UL que tenga el ID "botonera". Si queremos podemos obviar el UL y escribir directamente #botonera.

Tambien podemos diseñar a partir de una CLASS. La forma de seleccionar elementos en CSS con determinada CLASS es a partir del caracter . (punto). Se coloca el nombre de la etiqueta seguido el caracter punto y el nombre de la CLASS:

p.destacado {

font-size: 200%;

}

En este caso queremos diseñar todos los parrafos que tengan la clase destacado. Al igual que los ID se puede colocar solo el ".destacado" pero en este caso puede haber mas de un elemento con la misma CLASS y puede que no querramos seleccionarlos a todos.
